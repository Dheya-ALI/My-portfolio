MARBERT-based Sentiment Analysis for Arabic

This project focuses on enhancing the performance and accuracy of sentiment analysis tasks specifically for the Arabic language. The goal is to classify sentiments in Arabic text as positive, negative, or neutral, providing valuable insights into opinions expressed on social media.

  • Algorithm: MARBERT (a variation of BERT tailored for Arabic)
  • Data Source: Arabic text data harvested from Twitter

The model is evaluated using metrics such as accuracy, precision, recall, and F1 score. The results are visualized through confusion matrices, ROC curves, and sentiment distribution charts.

Password Strength Checker for Angular 6

The Password Strength Checker visually displays the strength of a password, offering immediate feedback to users. Utilizing the zxcvbn library, it assesses password security and provides a comprehensive evaluation, along with helpful suggestions and warning messages. This interactive tool encourages users to create stronger passwords by highlighting vulnerabilities and recommending improvements. By integrating visual indicators, the meter enhances the user experience and fosters better security practices, ultimately protecting sensitive information more effectively.

Recycling Trash (TRCHNES)

This project tackles Indonesia's plastic waste crisis by developing a web-based system for efficient trash management. Indonesia produces 7.8 million tons of plastic waste annually, with 4.9 million tons mismanaged, leading to severe environmental issues. The solution provides users with access to nearby garbage collection points, guides on trash segregation, and real-time notifications for trash collection. It aims to improve waste recycling, facilitate household waste management, and promote proper disposal practices. The project is available on GitHub.
